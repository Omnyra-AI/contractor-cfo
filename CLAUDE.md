# Contractor CFO

Marketing site for Contractor CFO services.

## Purpose

Landing page and marketing content for contractor-focused CFO services.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Vercel deployment

## Development

```bash
pnpm dev      # Start dev server
pnpm build    # Production build
pnpm lint     # ESLint
```

## Deployment

Auto-deploys to Vercel on push to main.
